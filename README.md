MyAwesomePongGame
==================
A Pong game in SDL 2

This is a single player game whare the player controll both paddles

Controls
--------
* Up - move up
* Down - move down

Requirements
------------
* SDL 2.0
* SDL_ttf

Font
----

Notes
-----
